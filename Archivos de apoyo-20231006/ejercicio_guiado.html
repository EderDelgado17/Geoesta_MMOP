Carga de datos

Primeramente, se cargan los datos, los espaciales y los no espaciales. En este caso, se trabajarn los resultados de pobreza que publica el CONEVAL (https://www.coneval.org.mx/Medicion/Paginas/Pobreza-municipio-2010-2020.aspx) y los archivos espaciales .shp que comparte INEGI (que aqu se los comparto yo).

Archivo no espacial

Ya que la informacin la comparte CONEVAL en un archivo no-espacial (tabla de Excel), utilizaremos el paquete llamado readxl para leer datos almacenados en hojas de clculo .xlsx.

Para leer archivos espaciales .shp, se utilizar la funcin read_sf del paquete sf.

*Se deben de tener instaladas la paqueteras instaladas previamente.

require(readxl)
dato_coneval<-read_xlsx("P:\\Geoesta\\Estadistica-Espacial-main\\coneval\\pobreza_2010_2015_2020.xlsx",sheet="Hoja1")

Ahora, leer el archivo espacial, el cual es tipo .shp. Explorar los polgonos y las features ligadas a cada polgono.

require(sf)
datos_muni<-read_sf("P:\\Mapas\\00mun2.shp")

Posteriormente, se har una unin de los dos archivos anteriores en solo uno. Por fines ilustrativos, solo se utilizar la columna pob_ing_extre_porc_2020 de la base de datos de CONEVAL. La funcin que une bases de datos, utilizando campos llave, se llama merge y est cargada en la versin base de R.

datos_esp_coneval<-sp::merge(datos_muni,dato_coneval[,c(3,139)],by.x="CVEGEO",by.y="cve_mun",all.x=TRUE)
datos_esp_coneval$pob_ing_extre_porc_2020<-as.numeric(datos_esp_coneval$pob_ing_extre_porc_2020)

El primer mapa a realizar, ser mediante la funcin ggplot de la paquetera ggplot2. El color con el que se rellenarn los polgonos (municipios) ser continuo con el tema tipo inferno de la funcin scale_fill_viridis_c

library(ggplot2)
library(ggspatial)
ggplot(data = datos_esp_coneval) +
  geom_sf(aes(fill = pob_ing_extre_porc_2020), alpha = 0.8) +
  scale_fill_viridis_c(option="B",direction = -1) +
  ggtitle(label = "Pobreza extrema % 2020", subtitle = "Por ingreso")

Con fines ilustrativos, se realizar otro mapa, en el cual se generarn cinco clases de la variable en inters.

library(RColorBrewer)
my_colors <- brewer.pal(9, "Reds")
